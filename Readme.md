# Kaos Operating System 

`For support 32 bit` 

### Requirements
* gcc++
* lib36-dev-i386

### For Execute 
* Virtualbox
* xorriso

### Make iso file

```
make install
```

### Execute Virtual Machine

```
make run
```
`Note:`
Please Virtualbox vmname change in line 38 Makefile

```
VirtualBox --startvm 'VMNAME' &
```
